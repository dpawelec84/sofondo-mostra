## Premium Features

Your support helps maintain and improve these premium features:

### Premium Recipes
- **Agency** - Dark theme with gradient effects and high visual polish
- **Product Launch** - Specialized landing page with high conversion focus
- **App Download** - Mobile app-focused landing with app store integration
- **Newsletter** - Clean, minimal style for newsletter signups

### SEO Suite
- Meta tags for search engines
- Open Graph for social sharing
- Twitter Cards for rich previews
- JSON-LD structured data
- Auto-generated XML sitemap

### Future Components
- Forms and validation
- Analytics integration
- Additional UI components

## Free Recipes

These recipes are always free:
- **Corporate** - Classic business template, great entry point
- **Startup** - Popular SaaS/startup landing page
- **Nonprofit** - Purpose-driven design for organizations

## Unlock Premium

Premium features are free to use with a gentle reminder to support. To skip future prompts:

```bash
npm run unlock:premium
```

Or press 's' when prompted during `npm run init:recipe`.

Thank you for your support!

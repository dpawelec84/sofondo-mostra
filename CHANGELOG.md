# Changelog

All notable changes to Sofondo Mostra will be documented in this file.

## [1.0.0] - 2024-12-15

### Added
- **Recipe System**: 7 pre-configured themes (corporate, startup, nonprofit, agency, product-launch, app-download, newsletter)
- **Premium Features System**: Pay-what-you-want model with unlock script and support prompts
- **SEO Suite**: Full SEO component with meta tags, Open Graph, Twitter Cards, structured data, sitemap, and robots.txt
- **Form Components**: Form, Input, and Button components with validation, honeypot spam protection, and multiple integration guides (Netlify, Formspree, Web3Forms)
- **Header Layouts**: 3 layouts (standard, centered, minimal) with mega menu navigation
- **Footer Layouts**: 4 layouts (grid-4col, flex-row, flex-sections, minimal-centered)
- **Logo Options**: 5 styles (text-only, icon-text, gradient-box, emoji, SVG icons)
- **Navigation**: React-based mega menu with Radix UI primitives
- **SubNav Component**: Sticky sub-navigation with simple and grouped variants
- **FadeInOnScroll**: Scroll-triggered animations
- **Smooth Scrolling**: Lenis-powered smooth scroll (opt-in)
- **Custom Scrollbar**: Theme-aware scrollbar styling (opt-in)
- **View Transitions**: Smooth page transitions (opt-in)
- **Dark Theme Support**: Full dark mode with CSS variables
- **Accessibility**: Skip link, proper heading hierarchy, color contrast compliance
- **Documentation**: Getting started, configuration, components, forms, SEO, customization, recipes, deployment, and troubleshooting guides

### Technical
- Astro 5.x static site generator
- React integration for interactive components
- Radix UI for accessible mega menus
- TypeScript configuration
- CSS variables for theming
- Mobile-first responsive design

## Version Guidelines

### Versioning
This project follows [Semantic Versioning](https://semver.org/):
- **MAJOR**: Breaking changes to configuration or component APIs
- **MINOR**: New features, components, or recipes
- **PATCH**: Bug fixes and documentation updates

### Categories
- **Added**: New features
- **Changed**: Changes to existing functionality
- **Deprecated**: Features to be removed in future
- **Removed**: Removed features
- **Fixed**: Bug fixes
- **Security**: Security patches
